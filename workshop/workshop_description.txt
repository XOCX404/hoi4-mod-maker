[h1]HOI4 Fantasy World Map Maker[/h1]
[b]v1.0.1 Beta[/b]

[h2]English[/h2]
A standalone desktop tool for creating complete HOI4 total-conversion mods from scratch — no manual file editing required.

[b]Features:[/b]
[list]
[*] Draw land, sea, and lakes with circular brush / fill / transform tools
[*] Auto-generate provinces (Poisson disk + Lloyd relaxation, supports density map)
[*] Smart heightmap generation (coastal distance field + Perlin noise mountains)
[*] Smart terrain assignment (height-based with noise blending)
[*] Auto-generate rivers from heightmap (flow accumulation)
[*] State / Country / Strategic Region / Continent management
[*] Railway / Supply Node / Adjacency (strait) editing
[*] One-click full MOD export (2000+ files, 5-second export)
[*] Import existing MOD maps
[*] Undo/Redo support
[*] Chinese / English bilingual UI
[/list]

[b]How to use:[/b]
After subscribing, find the tool folder in your Steam Workshop directory and run [i]hoi4_map_maker.exe[/i].
Typical path: [i]Steam\steamapps\workshop\content\394360\<id>\hoi4_map_maker.exe[/i]

[b]Supported map sizes:[/b] 2048x1024 / 3072x1536 / 4096x2048 / 5632x2048

[b]Requirements:[/b] Windows 10+, HOI4 1.17.x
